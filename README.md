# Project to Text Merger

Инструмент для быстрой сборки контекста вашего проекта в один файл, оптимизированный для работы с современными LLM.

# ✨ Что нового в версии 2.0
**Графический интерфейс (GUI):** Удобное дерево файлов с галочками — выбирайте только те файлы, которые нужны для текущей задачи, чтобы экономить токены.
- **AI-Оптимизация**:
  * Код оборачивается в XML-теги <file path="...">, что помогает ИИ лучше понимать структуру и границы файлов.
  * Автоматическая генерация Mermaid.js диаграммы для визуализации архитектуры проекта.
- **Markdown-формат:** Выходной файл .md с подсветкой синтаксиса внутри блоков кода.
- **Умная фильтрация:** Автоматический пропуск папок зависимостей (node_modules, venv), гита и бинарных файлов.

## 🚀 Как использовать

1. Запуск: Убедитесь, что у вас установлен Python 3.x, и запустите скрипт:
```Bash
python main.py
```

2. Выбор файлов:
* В открывшемся окне выберите папку вашего проекта (по умолчанию открывается текущая).
* Отметьте нужные файлы и папки галочками (Двойной клик или Пробел).
* Используйте кнопки "Выбрать всё" или "Снять всё" для быстрой настройки.

3. Генерация:
* Нажмите кнопку "СГЕНЕРИРОВАТЬ КОНТЕКСТ ДЛЯ ИИ".
* В корне вашего проекта появится файл `ai_project_context.md`.

4. Работа с ИИ:
Просто перетащите созданный файл в чат с ИИ или скопируйте его содержимое.

# 🛠 Требования
* Python 3.6+
* Библиотека tkinter (обычно идет в комплекте с Python). Если у вас Linux и возникла ошибка, установите её:
```Bash
sudo apt-get install python3-tk
```
